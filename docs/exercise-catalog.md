# Exercise Catalog Plan

## MVP Scope

MVP 1 should include around 50 built-in exercises.

The catalog should be grouped by muscle group and support localized labels for Russian and English.

## Post-MVP Scope

After MVP 1, the built-in catalog can expand toward roughly 150 exercises.

Custom exercises should be implemented in MVP 1 if feasible, or immediately after MVP 1.

## Muscle Groups

Initial muscle groups:

- Chest
- Back
- Legs
- Shoulders
- Biceps
- Triceps
- Core
- Glutes
- Cardio
- Full body
- Other

## Suggested MVP Distribution

This is a planning target, not the final catalog.

- Chest: 6 exercises
- Back: 7 exercises
- Legs: 8 exercises
- Shoulders: 5 exercises
- Biceps: 4 exercises
- Triceps: 4 exercises
- Core: 5 exercises
- Glutes: 3 exercises
- Cardio: 3 exercises
- Full body: 3 exercises
- Other: 2 exercises

Total: 50 exercises

## Exercise Tracking Modes

Each exercise should define which logging fields are shown.

Initial modes:

- Weight and repetitions.
- Bodyweight repetitions.
- Time.
- Distance and time.
- Weighted bodyweight.
